CATHI Tool Set-Up


Before you can start creating projects, you need to perform a set-up. During this set-up, the Conserved Domain Database is downloaded from the NCBI FTP site, along with genome assembly files utilized during BLAST database creation. Additionally to this setup, these files can be downloaded later on if there is a new release for GenBank or RefSeq.



The download and set-up process can take up to >20 minutes, depending on your internet connection.
